### 中文

#### 简介

2020年安卓原生练手项目，估计不会继续了，就当存个档吧。

完成部分 : 主页、下单、快速下单、购物车、账户管理

未完成的部分 : 评论区、搜索引擎、人工客服、商品图像检测、分享

#### 快速开始

删掉以下文件即为原项目，可直接导入Android Studio Dolphin(2021.3.1) : 
- read_me_img
- README.md
- .gitattributes
